伪代码书写规范

伪代码是一种接近自然语言的算法描述形式，目的是在不涉及具体编程语言的情况下将算法的流程和含义清楚的表达出来，因此没有一个统一的规范。

一般来说：

  1. 有一个标题和编号

  2. 明确输入和输出

  3. 赋值一般使用 <- 来表示

  4. A[i] 表示 A 的第 i 个元素，A[1...j] 表示从下标 1 到 j 的子数组

  5. 返回值使用 return 关键字

  6. 函数的调用使用函数名 + 传入参数的形式
